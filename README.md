# 조별활동 이탈 방지를 위한 조편성 전략 제시

## 1. 프로젝트 개요
BDA 학회원 조별활동 데이터를 활용해 조별활동 중 이탈에 영향을 미치는 요인을 조사하고, 
운영 관점에서 효과적인 조편성 개선안을 제안한다.

- 분석 단위: 학회원 단위
- 종속변수: group_leave(이탈 여부, 0/1)
- 데이터 기준 이탈 규모: 참여자 3,473명 중 510명(약 14.7%)

## 2. 사용 데이터
- member.csv: 학회원 정보 데이터
- group_study.csv: 조별활동 정보 데이터
- group_point.csv: 조별활동 점수 데이터
  
## 3. 분석 목표
1) 이탈과 유의미한 관련이 있는 요인 확인 - odds ratio, p-value를 산출해 정량적으로 비교
2) KMeans 군집분석으로 학회원 유형 분류 및 군집별 이탈률 비교  
3) 결과 기반 조편성 및 운영 개선안 제시

## 4. 전처리 및 변수 정의 요약
### 4.1 데이터 병합
조인 키(member_id, generation, class_code, group_number 등) 타입을 통일한 뒤 3개 데이터를 순차 Left Join하여 분석용 테이블(merged_data.csv)을 구성했다.

### 4.2 결측 처리
모델 입력 변수와 종속변수를 기준으로 결측 포함 행을 제거하여 학습 데이터를 구성했다.

### 4.3 주요 독립변수
- 개인 특성: IT 전공 여부, 직업(job), 재참여 여부(before_id 존재 여부)
- 활동/운영: 활동 방식(on_offline), 기수(generation), 활동월차(nth_month)
- 역할/성과: 조장 여부(group_master), 우수조 여부(excellent_group), 활동 점수(point)

### 4.4 다중공선성(VIF) 처리
VIF를 계산해 다중공선성을 확인하였고, Random Forest와 XGBoost 기반 중요도를 활용해 주요 변수를 선택하여 입력 변수 수를 축소했다.

## 5. 분석 방법
1) EDA: 가설 변수별 이탈률 비교 시각화  
2) 로지스틱 회귀: 계수/유의확률(p-value)/오즈비(OR)로 이탈 영향 요인 해석  
   - OR > 1: 이탈 위험 증가  
   - OR < 1: 이탈 위험 감소  
3) 군집분석: 표준화 후 KMeans(k=3)로 저/중/고위험군 분류  
4) 예측 성능 확인: XGBoost 예측확률에 대해 precision-recall curve를 기반으로 F1 값이 최대가 되는 최적 임계값을 산출하여 적용했다.

## 6. 핵심 결과 요약
### 6.1 EDA 기반 요약
- 직업별 이탈률: 직장인 35.7%, 대학생 15.2%, 취준생 11.5%, 대학원생 6.7%
- 오프라인 이탈률 8.4%가 온라인 18.1% 및 온오프라인 20% 대비 낮음
- 조장 이탈률 10.6%가 일반 조원 16%보다 낮음
- 활동 점수 높을수록 이탈 감소 경향

### 6.2 로지스틱 회귀 (유의수준 0.05)
오즈비 기준으로
- 온라인: 이탈 위험 증가 요인(OR 1.34)
- 조장여부: 이탈 방어 요인(OR 0.59)
- 활동점수: 이탈 방어 요인(OR 0.79)
- 기수: 이탈 방어 요인(OR 0.85)

통계적으로 유의하지 않은 요인은 보조 변수로 사용했다.

### 6.3 군집분석 결과
KMeans(k=3)로 저/중/고위험군을 구성했고, 군집별 이탈률은 8.8% / 11.1% / 22.7%로 차이가 확인되었다. 
고위험군은 온라인 비중이 높고 활동점수 등 성과 지표가 낮은 특성을 보였다.

## 7. 운영 적용 아이디어
1) 위험군 분산 조편성: 이탈 위험 요인을 가진 학회원이 특정 조에 집중되지 않도록 분산 배치  
2) 역할 기반 참여 유도: 조장에게 업무 집중을 막고 역할 세분화/순환 배정으로 책임 분산  
3) 온라인 참여자 전용 운영 장치: 주간 목표/산출물 템플릿, 주 1회 고정 비대면 회의, 질문방/사담방 분리 등으로 진행 불확실성 감소  
4) 사후 운영: 위험도 기반 모니터링 후 단계적 개입(리마인드→개인 안내→조 재정렬/멘토 투입), 초기 1개월 집중 케어 및 휴회기간 전후 인센티브 부여
